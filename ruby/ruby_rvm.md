# rvm

* インストールされている ruby のバージョンを表示  
rvm list
* 用意されている(インストール可能な) ruby のバージョン一覧を表示  
rvm list known
* バージョン指定して ruby をインストール  
rvm install 2.1
* 指定したバージョンの ruby を再インストール  
rvm reinstall 2.1
* 指定したバージョンの ruby をアンインストール  
rvm uninstall 2.1
* OS にインストールされている ruby を使う  
rvm system
* 指定したバージョンの ruby を使う  
rvm 2.2
* デフォルトのバージョンを指定  
rvm --default use 2.2  
(rvm default 2.2)
