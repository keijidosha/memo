### for

* 10回繰り返す
  ```
  10.times do |ii|
    ...
  end
  ```
  または
  ```
  for ii in 1..10 do
    ...
  end
  ```
* 1,3,5 だけ繰り返す
  ```
  for ii in [1,3,5] do
    ...
  end
  ```
  ii は 0 〜9
* 5〜10 まで繰り返す
  ```
  5.upto(10) do |i|
    ...
  end
  ```
* 10 => 5 まで繰り返す
  ```
  10.downto(5) do |i|
    ...
  end
  ```
* 5 から 25 まで、5単位で繰り返し
  ```
  5.step(25, 5) do |i|
    ...
  end
  ```
